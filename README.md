# Интерактив "Круг из семи маленьких иконок"

Автономный экран по ТЗ из Word-файла в этой папке.

## Запуск

Запустите `start-fullscreen.bat` для полноэкранного режима или откройте `index.html` в браузере. Сервер и интернет не нужны.

## Управление

- Нажатие на любую из семи денег открывает модальное окно с пояснением.
- Модальное окно закрывается кнопкой `Закрыть`, клавишей Escape или тапом по затемненному фону.
- Знак рубля расположен в центре как смысловой и визуальный центр композиции.

## Изображения

Карточки уже подключены к PNG-файлам в папке `assets`:

- `01-shell.png` - ракушка
- `02-hide.png` - шкура
- `03-ingot.png` - слиток
- `04-coin.png` - монета
- `05-jar.png` - кувшин
- `06-banknote.png` - ассигнация
- `07-card.png` - пластиковая карта

Чтобы заменить временные картинки на финальные, положите PNG с такими же именами в папку `assets`.
